### 云南旅游

昆明长水机场 -> 大理 -> 丽江 -> 香格里拉 -> 西双版纳

* 路线
  * day1: 抵达云南 -> 昆明 -> 昆明老街或滇池
    * 【住昆明】
  <!-- * day2: 昆明 -> `石林景区` -> 楚雄 -->
    <!-- * 1 小时车程 -->
  * day2: `洱海` -> 大理花海(网红打卡：天空之境、玻璃球) -> `大理古城`
    * 【住大理】
    * 租小车环洱海一圈大概一天左右。
  * day3: `玉龙雪山` + `蓝月谷` -> 丽水金沙 -> `丽江古城`
    * 【住丽江】
  * day4 `丽江千古情` -> 束河古镇 -> 茶马古道 -> 返回昆明
    * 【住昆明】
  * day5: 返程回家

* 大理: 「大理等着你客栈」
* 丽江：大研古城里
